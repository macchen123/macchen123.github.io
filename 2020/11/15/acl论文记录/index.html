ACL2020

一、CH-SIMS: A Chinese Multimodal Sentiment Analysis Dataset with Fine-grained Annotation of Modality

  • 单位

    • 清华大学

  • 文献来源

    • ACL2020

  • 论文子方向

    • 多模态情感分析

  • 问题

    • 以往的数据集只包含统一的多模态情感标注，但是这种标注不能很好的反映单模态的情感类别。
    • 不能够捕获到模态之间的差异性。
      • 统一的多模态标注可能对于模型学习单模态的特征信息时造成误导。

  • 本文工作

    • 构建了一个中文多模态情感分析数据集——CH-SIMS；
    • 提出了一个基于后端融合的多模态多任务学习框架。

  • 本文目的

    • 通过该数据集，研究者既可以做多模态的情感分析任务，也可以做单模态的情感分析任务。
    • 研究者也可以在多模态情感分析任务上通过用独立的单模态标注来研究新的方法。

  • 数据集

    • SIMS有文本，语音，视频3个模态，并且除了有统一的多模态情感类别标注以外，对每个单模态进行了情感类别标注。

    • 视频片段来自于国产电影，国产电视剧以及各种国产综艺节目。

      其中M表示多模态情感标注，T表示文本情感标注，A表示音频情感标注，V表示视频情感标注。

  • 本文方法

    • 后端融合是将不同模态数据分别训练好的分类器输出打分进行融合。这样做的好处是，融合模型的错误来自不同的分类器，而来自不同分类器的错误往往互不相关、互不影响，不会造成错误的进一步累加。

    • 利用单模态和多模态的标注，本文构建了一个多模态多任务学习框架。

      • 在框架内，单模态与多模态任务在底部共享特征表示子网络。

      • 我们引入了三种后端融合模型TFN,LMF,LF-DNN来对多模态数据建模。

  • 实验表现

举例说明：LF-DNN是对统一的多模态标注进行多任务训练。MLF-DNN是LF-DNN的扩展，它相比前者还引入了单模态标注对单任务的训练，形成了“单模态情感标注+多模态情感标注”模型。这样使得实验表现显著提升。

  • 不足

    • 在多任务学习中，子任务的异步性可能会对多模态情感分析造成不良影响。
      • 异步性：我们首先对多模态标注用后端融合网络进行多任务学习，然后引入了一个单模态单任务学习。相当于这个单模态被学习了两次（反向传播时受多模态损失和单模态损失的影响），这样会导致异步性。
      • 如果可以引入很多单模态单任务，这个负面效果渐渐会被抵消。

  • 展望

    • 探索多模态与多任务之间的联系。
    • 引入更多的融合策略：前端融合、中期融合。

二、Improving Multimodal Named Entity Recognition via Entity Span Detection with Unified Multimodal Transformer

  • 单位

    • 南京科技大学
    • 新加坡管理大学
    • 新加波DBS银行

  • 文献来源

    • ACL2020

  • 论文子方向

    • 多模态命名实体识别

  • 问题

    • 现有的方法在识别正式新闻文本中的实体方面比较成功。但由于其长度短且噪音大，大多数实体在非正式社交媒体文本(如推文)中表现不佳。
    • 非语境化的单词表示，其中每个单词由相同的向量表示，而不管它出现在哪个语境中。然而，社交媒体帖子中许多多义实体的含义往往依赖于其上下文词
    • 虽然大多数现有方法集中于对模态间交互进行建模以获得单词感知的视觉表示，但是它们最终隐藏层中的单词表示仍然基于文本上下文，而文本上下文对视觉上下文不敏感。
    • 大多数以前的方法在很大程度上忽略了包含视觉信息的偏差。事实上，在大多数社交媒体帖子中，相关图像往往只突出句子中的一两个实体，而不提其他实体。在这些情况下，直接整合视觉信息将不可避免地导致模型更好地识别图像突出显示的实体，但无法识别其他实体。

  • 本文工作

    • 针对MNER设计了一个多模态Transformer模型。
    • 基于这个模型设计了一个统一架构。

  • 本文目的

    • 面向(非正式)社交媒体帖子的多模态命名实体识别。

  • 数据集

    • 两个公开的MNER数据集。

      • 分别是推特14-15年和16-17年的用户推文（帖子）数据（推文格式是多模态）。

      • 本文将这两个数据集分别命名为’TWITTER-2015’,’TWITTER-2017’。

  • 本文方法

    • 针对MNER设计了一个多模态Transformer模型。

      • 该模型为Transformer提供了一个多模态交互模块（MMI）来捕捉词和图像之间的模态间动态。

    • 基于这个模型设计了一个统一架构。

      • 基于文本的实体跨度检测模块（text-based entity span detection module）。它用来减少上述模型中视觉环境中的误差偏置。

  • 实验表现

  • 不足

    • 在社交平台的帖子中，对于不匹配的图片和文本构成的帖子的情况下， UMT（统一多模态Transformer）的表现并不好。
    • 现在公开可用的MNER数据集相对较小。

  • 展望

    • 提高UMT的性能让它可以动态的过滤掉图片中存在的潜在噪声。
    • 计划构建一个基于多种社交平台的帖子的未标记数据集，提出一个有效的框架，将它们与少量已注释数据相结合，以获得性能更好更稳定的MNER模型。

三、Reasoning with Multimodal Sarcastic Tweets via Modeling Cross-Modality Contrast and Semantic Association

  • 单位

    • 中科院
    • 中科大

  • 文献来源

    • ACL2020

  • 论文子方向

    • 多模态推理

  • 问题

    • 传统的基于文本的方法不足以检测多模态讽刺推文。

  • 本文工作

    • 提出了一个新的方法来建模在相关语境下的跨模态对比。

  • 本文目的

    • 为了推理多模态讽刺推文。

  • 数据集

    • Twitter上hashtag为#sarcasm, #sarcastic, #irony, #ironic等的推文。

  • 本文方法

    • 构造了一个分解与关系网络（Decomposition and Relation Net）。

      • 分解网络通过表示推文中的图片和文本的共性与差异来隐式建模跨模态对比信息。
        • 因为短文本通常没有足够的语境信息，所以我们需要结合对应的图片来推理推文是否带有嘲讽。
  • 关系网络通过一个跨模态注意力机制来显式捕获图片和文本的语义关联信息。


解释：ANP指形容词-名词对，它是将推文文本进行分词，筛选出名词和形容词，得到每个词的词向量，然后计算图片与（形容词-名词）之间的相似度筛选出前5个最接近的ANP，一并输入到模型中。

  • 实验表现

    • Attribute表示名词。
