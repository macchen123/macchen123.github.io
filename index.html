概率（第二周）

2.1 概率公理性质

  • 公理（Axioms）

    • 近代数学常以数条公理作为整套理论的基石
      • ex：线性代数的8条公理
    • 公理常是不能被证明的基本性质
    • 公理常是非常基本的性质（所以不是废话）
    • 公理越少条、公理越基本的数学学科越厉害！

  • 概率三公理（Axioms of Probability）

    • 公理1：对任何事件.. 而言, P(A) ≥0。
    • 公理2：P(S)=1
    • 公理3：事件A1,A2,…互斥 => P(A1∪A2∪A3∪…) = P(A1)+P(A2)+P(A3)+…
    • 公理3搭起了集合运算与概率运算的桥梁！
    • 后面的性质都可以用这3条公理证明。

  • 衍生的概率性质

    • ex：从一副52 张扑克牌抽中一张，结果为Ace 之概率为何？

    • 这个性质的证明用到了公理3以及上面刚提到的两个性质。

      Ex：在大陆随便碰上一个人，此人爱甜豆花或爱咸豆花概率为何？

2.2 条件概率

  • 概率常反映我们对某些事情的了解程度

    • 当发生这种情况时：

      正确答案就只有B，D了，所以当混哥看到卷哥试卷露出的线条后，想要知道B，D为正确答案的概率就是指条件概率。

  • 条件概率表示

    • P( 𝑿| 𝒀 )

      𝑿：所关心之事件

      𝒀 ：条件（观察到的,已发生的事件）

    • 针对上例：

  • 条件概率计算（还是以上例为例子）

    • 𝑷( B 为正解｜卷矫漏曲) = ?（卷矫漏曲：卷哥矫情堵答案却无意漏出了答案的线条）

      • 未偷看时，正确答案未知。样本空间为：

      • 卷矫漏曲后，新的样本空间变为：

      • 由于发生了卷矫漏曲，所以以下情况就不可能发生了，即：

        𝑷( A 为正解｜卷矫漏曲) = 𝑷 (𝑪 为正解｜卷矫漏曲) = 0

        • 延伸：若某实验结果 𝒐𝒊 与某条件 𝒀 不相交，则𝑷 (𝒐𝒊 |𝒀) = 0

      • 不管卷矫漏曲发生否，「B为正解」与「D为正解」的概率比例应该一样，故：

        𝑷( B 为正解｜卷矫漏曲)：𝑷 (𝑫 为正解｜卷矫漏曲) = 𝑷( B 为正解)：𝑷 (𝑫 为正解)

        卷矫漏曲后只有可能出现「B 为正解」或「D 为正解」，故：

        𝑺 ′ = {𝑩, 𝑫} , 𝑷( B 为正解｜卷矫漏曲) + 𝑷 (𝑫 为正解｜卷矫漏曲 )= 𝟏

        根据上述二式我们得到:

        卷矫漏曲 = {B 为正解，D 为正解}

    • 延伸：若某条件事件 Y 包含数个实验结果：Y = { {o₁,o₂,…,on} }

      • P(oᵢ|Y) = P(oᵢ)/( P(o₁)+P(o₂)+ … + P(on) ) = P(oᵢ)/P(Y)
      • Ex: 考虑某事件 X = {o₁,o₂,q₁,q₂} , 已知条件事件 Y = {o₁,o₂,o₃} 发生了，则
        • P(X|Y) = P(o₁|Y) + P(o₂|Y) = P(o₁)/P(Y) + P(o₂)/P(Y) = P({o₁,o₂})/P(Y)
        • 因为Y已经发生了，而q1，q2不在Y里面，所以压根不用考虑它俩。

    • 终极延伸: 若已知某条件事件 Y 发生了，则对于任何事件 X ，条件概率如下：

      • P(X|Y) = P(X∩Y)/P(Y)

  • 条件概率的性质

    • 性质1,2,3分别与概率3公理对应。

  • Total Probability 定理（全概率公式）

    • 其实是切面包定理的条件概率版本。

  • 贝叶斯定理 (Bayes’ Rule)

    • 我们有时候需要转换条件事件和关注事件，比如对于阿宅来说，条件事件是店的生意的好坏，关注事件是店员是否对他笑。

      而对于老板来说，条件事件是店员是否对他笑，关注事件是店的生意的好坏。

    • 证明借助了条件概率（左边）与 Total Probability 定理（全概率公式）（右边）。

概率第一周

概率（第一周）

1.1 概率概论

概率范例

  • 丢铜板看到正面概率为 0.52
  • 明天下雨概率为 60%
  • 丢四颗骰子得到一色的概率为 1/216 （四个骰子投到同样的点数）

概率的本质

  • 概率 = 0.6 代表什么意思？

  • 在回答这个问题前我们先问：

    • – 距离 = 1.23 公尺是什么意思？

      表示1.23*(1/299792458)秒光走的距离

    • – 时间 = 8.2 秒是什么意思？

      表示8.2乘以919263170倍的铯原子的震荡周期

  • 如何理解概率 = 0.6？

    • 我们借助幸运之轮来理解，它是美国一个节目上的游戏，参赛者转到哪个刻度上，就会获得相应的奖品。

    • 我们将幸运之轮的周长定为1，然后到0.6标记一个刻度，所以一件事情发生的概率=0.6可以理解为可能性跟幸运之轮【X停在长度为0.6的红边上】这件事发生的概率是一样的。

      以下这两个图都表示概率=0.6。

为什么要研究概率

  • 主要原因是我们人类太渺小，很多事情的运作都不是我们能掌握的。
  • 人类其实是逃避现实的，有些事物它注定是那样发生的，但是由于我们无法掌握，就只能借助概率来帮助我们理解这个世界的运作。
  • 举例：比如一个差生，他在考试时面对ABCD选项时只能蒙一个，那么每个选项的概率是1/4。事实上，只有一个正确答案，它是必然要被选择的（概率为1），其他3个答案是不可能被选的（概率都为0）。
  • 我们在面对这个世界，其实有很多事情是我们没有办法掌握的 不管是它太复杂 以至于没有办法掌握 或是它本身就有随机性（抽扑克牌）让我们没有办法掌握为了掌握这些没有办法掌握的事情，我们才发展出概率来帮助我们理解这个世界，观察这个世界。

概率与统计的差异

  • 概率：概率模型已知，要学会怎么算某些事件的概率。
    • 已知一骰子为公平骰（即掷出每一点的概率都是1/6），看到偶数的概率为何？
  • 统计：概率模型未知，要学会怎么从大量的实验结果中去建立概率模型。
    • 不知一骰灌铅（作弊的骰子）否，欲知各点出现之概率模型？

1.2 集合论

名词复习

  • 元素（Element）

    • ex：小黑、小冀、小湘、小鄂、小美

  • 集合（Set）

    • ex：咸豆腐脑党A=｛小黑，小冀｝
    • ex：甜豆腐脑党B=｛小湘，小鄂｝

  • 子集（Subset）

    • ex：嫌咸党C=｛小湘，小鄂，小美｝

      B是C的子集，表示为：B⊂C

  • 全集（Universal Set）

    • ex：S=｛小黑，小冀，小湘，小鄂，小美｝

  • 空集（Empty Set）

    • ex：Ø=｛｝

  • 交集（Intersection）

    • ex：喜欢甜豆腐脑 且 喜欢咸豆腐脑者=｛｝=Ø

      𝑨 ∩ 𝑩 = { } = Ø

  • 并集（Union）

    • ex：喜欢甜豆腐脑 或 喜欢咸豆腐脑者=｛小黑，小冀，小湘，小鄂｝

      𝑨 ∪ 𝑩 = {黑, 冀, 湘, 鄂}

  • 补集（Complement）

    • ex：嫌咸党C=咸豆腐脑党A的补集

      C=$A^C$

  • 差集（Difference）

    • X-Y=｛在X但不在Y中的东西｝
    • ex：嫌咸党-甜豆腐脑党=C-B=｛小美｝

  • 不相交（Disjoint）

    • 如果𝑿 ∩ 𝒀 = 𝝓 → 𝑿, 𝒀
    • ex：甜党⋂咸党={ }，亦即甜党、咸党不相交!

  • 互斥（Mutually Exclusive）

    • 若一群集合 𝑿𝟏,𝑿𝟐, … ,𝑿𝒏 中任选两个集合 𝑿𝒊 ,𝑿𝒋 都不相交，则我们称 𝑿𝟏,𝑿𝟐, … ,𝑿𝒏 这群集合互斥
    • ex：甜党、咸党、小美党，三者两两不相交，故三者互斥!

De Morgan’s Law 定理

  • 证明思路：你中有我，我中有你(我是你的子集，你是我的子集，那么咱们相等)

  • ${(𝑨 ∪ 𝑩)}^C$=$A^C∩B^C$

1.3 概率名词

  • 实验（Experiment）

    • 我们用做实验来得到随机的，不确定的事情的结果

    • 一个概率实验包含了：步骤（procedures）、（概率）模型（model）、观察（observations）

    • 公平骰子已经暗示了概率模型是什么（1/6）

  • 结果（Outcome）

    • 实验可能的结果（做实验必须得有结果，否则没有意义）

  • 样本空间（Sample Space）

    • 概率实验的所有可能的结果的集合（全集），通常用S来表示
    • 同理：如果幸运之轮转3次，那么S=[0,1)x[0,1)x[0,1)，其实就是在三维坐标轴上，边长为1的正方体的体积。

  • 事件（Event）

    • 对实验结果的某种叙述。
    • 概率就是指实验结果符合某事件叙述的机会多大。
    • 在数学上，「事件」可以看成是「结果」的集合，亦即是 「样本空间」的子集。

  • 事件空间（Event Space）

    • 「事件空间」是包含所有事件的集合（集合的集合）
    • 一定要留意空集。
    • 概率是一个函数，其自变量是事件
      • P(事件) = 0.6 ⇒ 概率函数的自变量是：事件！
    • 概率可以看成是一个映射
      • 概率函数是从「事件空间」映射到 [𝟎, 𝟏] P:「事件空间」→ [𝟎, 𝟏]
